DataProcessing
==============

The ``DataProcessing`` module provides functionality for handling data loading and preparation. It includes methods to load data from CSV files, which can be useful in various data science projects. The primary method of this class is designed to simplify the process of reading data files and converting them into Pandas DataFrames for further analysis.

Overview
--------

This class contains static methods to load CSV files into DataFrames, allowing for flexibility with delimiters and decimal characters. By using this module, you can easily handle various CSV formats, regardless of the separator or decimal format used in the file. It simplifies the data preparation process, which is often the first step in any data analysis workflow.

Features
--------
- **Flexible CSV loading**: Supports loading CSV files with customizable separators and decimal characters.
- **Pandas Integration**: Converts the CSV data directly into a Pandas DataFrame, making it ready for analysis.
- **Ease of Use**: Provides a straightforward method for importing data, with sensible default parameters for most common cases.
