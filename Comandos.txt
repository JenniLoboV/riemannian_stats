Paso 1: Abrir la Terminal o Consola de Comandos
Primero, necesitas abrir la terminal en tu sistema operativo o usar la consola integrada en PyCharm, como mencionamos antes.

Paso 2: Activar el Entorno Virtual (si es aplicable)
Si estás usando un entorno virtual (recomendado para manejar dependencias de proyectos específicos), asegúrate de activarlo antes de instalar los paquetes. Esto asegura que los paquetes se instalen en el entorno virtual y no globalmente. Aquí te muestro cómo activarlo en Windows:

bash
Copy
.\venv\Scripts\activate
En PyCharm, generalmente, el entorno virtual se activa automáticamente cuando abres la terminal.

Paso 3: Instalar los Módulos con pip
Con el entorno activo, puedes instalar los paquetes ejecutando el siguiente comando en tu terminal o consola de comandos:

bash
Copy
pip install umap-learn pandas numpy matplotlib
Este comando instalará las últimas versiones de umap-learn (para UMAP), pandas (para manipulación de datos), numpy (para cálculos numéricos), y matplotlib (para visualización de datos).
